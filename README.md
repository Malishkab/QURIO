# QURIO - AI Campus Assistant

QURIO is an AI-powered campus assistant designed to simplify access to academic and administrative information for students. It serves as a centralized chatbot that provides instant answers, voice-based interaction, location-aware navigation, and real-time college updates through an intuitive web interface.

## Features

- AI chatbot for answering student FAQs
- Internship, research, and hackathon recommendations
- Voice-based interaction for improved accessibility
- Location-aware campus navigation
- Email integration for processing academic notifications
- Real-time updates on schedules, events, and announcements
- User-friendly web interface

## Tech Stack

- Python
- Flask
- HTML, CSS, JavaScript
- Firebase
- Google Gmail API
- OpenStreetMap
- Speech Recognition

## Workflow

1. Student submits a query through the chatbot.
2. The system identifies the user's intent.
3. Relevant information is retrieved from the knowledge base or integrated services.
4. Voice and location modules are invoked when required.
5. Responses are displayed through the chatbot interface.
6. Email and notification services provide real-time academic updates.
